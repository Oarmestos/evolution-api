# Identidad de la Plataforma - Avri

## 🚀 Misión
**Avri** es una plataforma avanzada de automatización de WhatsApp diseñada para gestionar comunicaciones, ventas y servicios de forma inteligente y eficiente.

## 💎 Marca
- **Nombre:** Avri
- **Concepto:** Agilidad, Conectividad y Asistencia Virtual Inteligente.
- **Uso:** El nombre debe utilizarse en todas las comunicaciones con clientes y en los títulos de la interfaz de usuario.

## 🛠️ Componentes Técnicos (Bajo el capó)
- **Motor:** Evolution API (v2.3.7)
- **Base de Datos:** PostgreSQL
- **Interfaz de Gestión:** Avri Manager (Puerto 8080)
